## Chapter01 

> 리액티브 프로그래밍 개요

* 함수형 프로그래밍 
우리가 기존에 접했던 절차지향, 객체지향 프로그래밍과 달리, 함수형 프로그래밍은 유기적 연결 및 동작이 프로그램의 최우선이 되는 프로그래밍 방식이다. 즉, 함수가 일급객체로써의 의미를 가지는 것이다. 

* 리액티브 프로그래밍
데이터 흐름과 전달에 관한 프로그래밍. 함수형 프로그래밍 언어의 도구들을 가지고 데이터 흐름을 composable(구성가능)하게 구현하는 것이다.
